# teaser

Teaser site

## Project Dependencies

This project is built with:

1. [NodeJS](http://nodejs.org)
2. [Gatsby](https://www.gatsbyjs.org/)

## Running the Project

**Development Mode**

```sh
$ git clone https://github.com/asucis/teaser.git
$ cd teaser
$ yarn
$ gatsby develop
```

Starts the site at `http://localhost:8000`!

**Production Mode**

```
$ gatsby serve
```
